# Smap-menu
## 设计目标
1. menu所有功能集成在smap逻辑内
2. 与玩家端app的逻辑接入。
3. PGC的定制化需求在这套系统中可良好拓展。
4. 整套交互细腻流畅生动。

 ## 分支管理 👻
 * main：主干分支，所有完成的新功能都合入此分支；
 * release：当前最新的发布版本，有热更的bug需要在此版本修复；
 * feature-xx：以特性开设的功能分支，功能完成后合入main，并删除此分支；
 * dev-xx：以开发者命名的分支，用于单人的小特性开发，合入main后，可以长期保存。
